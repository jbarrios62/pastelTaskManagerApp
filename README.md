# Pastel Task Manager App

A cute pastel desktop to-do list widget built with HTML, CSS, JavaScript, and Electron.

## Features

* Add, edit, complete, and delete tasks
* Frameless desktop window
* Always-on-top mode
* Pin/unpin toggle
* Draggable custom title bar
* Responsive layout for resizing
* Custom goodbye screen before closing
* Pastel glass-style UI

## Tech Stack

* HTML
* CSS
* JavaScript
* Electron
* Node.js

## How to Run Locally

Clone the repository:

```bash
git clone https://github.com/jbarrios62/pastelTaskManagerApp.git
```

Move into the project folder:

```bash
cd pastelTaskManagerApp
```

Install dependencies:

```bash
npm install
```

Start the app:

```bash
npm start
```

## Project Purpose

This project started as a simple browser-based task manager and was converted into a desktop productivity widget using Electron. It was built as a small developer-day project to practice desktop app development, responsive UI design, and GitHub project publishing.
